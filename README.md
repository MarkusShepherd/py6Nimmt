## Description

6 Nimmt! cardgame.
You can read rules in https://en.wikipedia.org/wiki/6_Nimmt!#Rules

## System requirements

* Python3


## Instructions
### Installation

```shell
git clone https://github.com/juanjo78git/py6Nimmt.git
cd py6Nimmt
$ pip install -e .  # for developers
or
$ pip install .
```

Or run without install:

```shell
$ python -m py6Nimmt.py6Nimmt
```
